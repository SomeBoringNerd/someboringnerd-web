FeurBot

The FeurBot is a minecraft tool with many features for anarchy players. The two big features are the /tpyes bot, for servers like 0b0t, to tpa anyone within a whitelist to your base, and a "auto stash" system, inspired by the "gringotts" system of the SpawnMasons (currently WIP).
The bot is currently not public, but once the bot will be done, it will be public, free, and open source.
The bot is meant to be run inside a headless minecraft instance, and the code is optimized to take as little resources as realisticly possible
